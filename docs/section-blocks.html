BlocksLink

Blocks are Gutenberg Blocks that can be used in WordPress Examples may include buttons, audio players, and callouts.

Blocks should be further categorized by a sub-category such as: Global, Homepage, Edition, Article corresponding to how they are used and where they are available to users in the theme. Such as restricted by post type.

Blocks GlobalLink

These are blocks that are available throughout the site on multiple post types and page templates

Global Aside BlockLink

Ready New Issue

A block for Aside tag type content, like a short bio on a Professor in the story, or a small explainer about subatomic particles in a longer story about physics. This is a parent block that allows specific child blocks, like image block, headline block, paragraph block, button block, etc. which can be added or removed as necessary.
