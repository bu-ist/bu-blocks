Edition Instagram Photo BlockLink

In Progress New Issue

A block that displays square photos from Instagram. Block supports either 1 image & caption, or a mini gallery with multiple photos and a Title for the overall gallery.

Examples Source: blocks/instagram-photo/style.scss, line 15 View Source
Default styling
.is-style-border

Border box style

.is-style-shadow

Drop-shadow box style

.is-layout-gallery

Gallery layout

Markup blocks/instagram-photo/bu-blocks-block-instagram-photo.hbs 
<aside class="wp-block-instagram-photo {{modifier_class}}">
			<div class="wp-block-instagram-photo-media">
				<figure class="wp-block-instagram-photo-image">
					<img src="https://placekitten.com/600/600">
				</figure>
			</div>
			<div class="wp-block-instagram-photo-content">
				<div class="wp-block-instagram-photo-caption">
					{{#unless modifier_class }}
						<p><strong>Lorem ipsum dolor</strong> sit amet, consectetur adipiscing elit. Maecenas massa augue, sodales nec convallis in, viverra in tortor.</p>
					{{/unless}}
		
					{{#ifContains modifier_class 'is-style-shadow' }}
						<p><strong>Lorem ipsum dolor</strong> sit amet, consectetur adipiscing elit. Maecenas massa augue, sodales nec convallis in, viverra in tortor.</p>
					{{/ifContains}}
		
					{{#ifContains modifier_class 'is-style-border' }}
						<p><strong>Lorem ipsum dolor</strong> sit amet, consectetur adipiscing elit. Maecenas massa augue, sodales nec convallis in, viverra in tortor.</p>
					{{/ifContains}}
		
						{{#ifContains modifier_class 'is-layout-gallery' }}
							<div class="wp-block-instagram-photo-gallery-title">
								<h3>Students Instagram Winter Wonder</h3>
							</div>
						{{/ifContains}}
				</div>
				{{#ifContains modifier_class 'is-layout-gallery' }}
					<div class="wp-block-instagram-photo-gallery-nav">
						<a href="" class="nav-next-icon"><span class="previous-nav-label">Next</span></a>
						<a href="" class="nav-prev-icon"><span class="previous-nav-label">Previous</span></a>
					</div>
				{{/ifContains}}
			</div>
		</aside>
